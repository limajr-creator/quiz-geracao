## 📌 Sobre o Projeto

Este projeto reúne duas experiências interativas desenvolvidas para treinamentos sobre **Conflito Geracional**, criadas para apoiar ações de **Treinamento & Desenvolvimento (T&D)** na **Revemar Academy**.

O objetivo foi transformar conteúdos tradicionais de treinamento em experiências digitais mais envolventes, modernas e emocionalmente conectadas com os participantes.

O projeto é dividido em:

* 📘 **Guia Interativo para Instrutores**
* 🎯 **Teste de Gerações Interativo (“Qual geração vive em você?”)**

O projeto busca unir:

* aprendizagem experiencial,

* tecnologia web,
* Inteligência Artificial aplicada ao desenvolvimento humano.

# 🎯 Teste Interativo — “Qual Geração Vive em Você?”

## 🧠 Sobre

O teste foi desenvolvido como uma dinâmica de abertura para treinamentos de Conflito Geracional.

A proposta é gerar:

* identificação emocional,
* engajamento imediato,
* quebra de gelo,
* reflexão sobre diferenças geracionais,
* introdução prática ao tema do treinamento.

O participante responde perguntas rápidas sobre:

* aprendizagem,
* comunicação,
* comportamento profissional,
* adaptação tecnológica,
* valores no trabalho.

Ao final, recebe um resultado geracional com:

* perfil comportamental,
* descrição personalizada,
* música representativa da geração,
* integração com Spotify,
* mensagem final personalizada.

* Baby Boomers
* Geração X
* Millennials
* Geração Z
* Geração Alpha

Também existe possibilidade de:

* resultado híbrido entre gerações,
* múltiplos perfis comportamentais.

# 🎯 Interactive Quiz — “Which Generation Lives Inside You?”

## 🧠 About

The quiz was designed as an opening activity for Generational Conflict training sessions.

Its purpose is to promote:

* emotional identification,
* immediate engagement,
* ice-breaking interaction,
* reflection on generational differences,
* a practical introduction to the training topic.

Participants answer quick questions about:

* learning styles,
* communication,
* professional behavior,
* technological adaptation,
* workplace values.

At the end, they receive a generational result featuring:

* behavioral profile,
* personalized description,
* representative song of the generation,
* Spotify integration,
* personalized final message.

---

## 🧬 Featured Generations

The quiz identifies tendencies related to the following generations:

* Baby Boomers
* Generation X
* Millennials
* Generation Z
* Generation Alpha

It also supports:

* hybrid generation results,
* multiple behavioral profiles.
